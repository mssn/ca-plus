This README is used to introduce our released datasets and source codes by our CoNEXT'23 work: 
“Dependent Misconfigurations in 5G/4.5G Radio Resource Control”.

We have conducted a reality check on dependent misconfiguration with 4.5G/5G datasets from three US operators and one China operator. We use one 5G phone model (Google Pixel 5) to collect two US datasets in the metropolitan areas A and B over three top-tier carriers. We also use a public dataset with 4G traces collected from one operator in China.

This reality check has covered 13,687 cells (4G: 12,778 and 5G: 899) and 340K signaling
messages. In this study, all three US operators run 5G in a non-standard-alone (NSA) mode. We have collected data speed results through file downloading experiments in the US.

2) Code Descriptions

-------------------------------------------------------------------------------
code/model_generator: 
Implement CSM and DSM models, and output configuration traces (dataset/config/csm) or delta configuration traces (dataset/config/dsm) for input logs (dataset/raw_logs).

Command:
$ python3 *_generator.py {input file path} > {output file path}
-------------------------------------------------------------------------------
code/misconfig_detector: 
Detect and output misconfiguration instances (D1-D5).

Command:
$ python3 misconfig_*.py {input file path} {output folder}
-------------------------------------------------------------------------------

3) Dataset Descriptions

-------------------------------------------------------------------------------
dataset/raw_logs:
Records the information of all parsed signaling messages, including serving cells, configuration and RSRP/RSRQ. dump_{area}.txt files are used for D4 instance analysis, and mmlab_{area}.txt files are used for the rest of instances.
-------------------------------------------------------------------------------
dataset/config:
The configuration traces generated by csm_generator.py and the delta configuration traces generated by dsm_generator.py.
-------------------------------------------------------------------------------
dataset/misconfig:
The list of detected misconfiguration instances. For each instance, information such as operator, PCell id, PCell channel, and misconfigured channels are included.
-------------------------------------------------------------------------------
